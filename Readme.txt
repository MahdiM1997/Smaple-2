# 🏥 Medical Portal Project
(محتویات این فایل را در  یک فایل نوت پد باز کنید اگر فونت فارسی قابل خواندن نیست)

-پروژه پورتال پزشکی یک پروژه Full-Stack شامل بک‌اند Node.js و فرانت‌اند مدرن (Vite + React) است.  در این پروژه که بیشتر بر backend ان تمرکز شده از تکنولوژی هایی مثل bcrypt cors dotenv express formidable jsonwebtoken mongodb و reduxjs/toolkit axios jwt-decode react-redux react-router-dom استفاده شده است.


## ✨ ویژگی‌های کلیدی
-مدیریت پزشکان و بروز رسانی اطلاعات انها
-مدیریت بیماران و پرونده‌های پزشکی و بروز رسانی اطلاعات انها
- احراز هویت کاربران (پزشک/مدیر سیستم)
- آپلود و مدیریت اسناد پزشکی    
- رابط کاربری مدرن و واکنش‌گرا
- API RESTful با امنیت JWT
-برای هر کاربر سطح خاصی از دسترسی تغیین شده

## 🚀 راهنمای شروع سریع

### پیش‌نیازها
- [Node.js](https://nodejs.org) v18.x یا بالاتر
- [npm](https://www.npmjs.com/) v9.x یا بالاتر
- [MongoDB](https://www.mongodb.com/) (محلی یا Atlas)

```bash
# 1. کلون کردن ریپازیتوری
git clone https://github.com/MahdiM1997/Sample-2.git
cd Sample-2

# 2. نصب وابستگی‌های بک‌اند
cd backend
npm install

# 3. نصب وابستگی‌های فرانت‌اند و راه اندازی محیط توسعه
cd ../frontend
npm install
npm run dev

# 4. راه‌اندازی محیط توسعه
cd ../backend
npm run start

_________________________________________________________________________________
-The Medical Portal project is a Full-Stack project including a Node.js backend and a modern frontend (Vite + React). This project, which focuses more on the backend, uses technologies such as bcrypt, cors, dotenv, express, formidable, jsonwebtoken, mongodb, and reduxjs/toolkit, axios, jwt-decode, react-redux, react-router-dom.

## ✨ Key Features
-Manage doctors and update their information
-Manage patients and medical records and update their information
- User authentication (doctor/system administrator)
- Upload and manage medical documents 
- Modern and responsive user interface
- RESTful API with JWT security
- Specific access level for each user

## 🚀 Quick Start Guide

### Prerequisites
- [Node.js](https://nodejs.org) v18.x or higher
- [npm](https://www.npmjs.com/) v9.x or higher
- [MongoDB](https://www.mongodb.com/) (local or Atlas)

```bash
# 1. Clone the repository
git clone https://github.com/MahdiM1997/Sample-2.git
cd Sample-2
# 2. Install backend dependencies
cd backend
npm install
# 3. Install frontend dependencies and start development environment
cd ../frontend
npm install
npm run dev
# 4. Start development environment
cd ../backend
npm run start
